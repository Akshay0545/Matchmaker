An internal matchmaking tool for The Date Crew — helps matchmakers manage clients, view compatibility scores, and send curated introductions.

## Tech Stack

**Frontend:** React 18 + Vite, Tailwind CSS, React Router, Axios, React Hot Toast, Lucide Icons

**Backend:** Node.js + Express, JWT authentication, JSON flat-file storage (customers, match pools)

**AI:** Groq API (Llama 3.1 8B Instant) for match scoring and notes analysis

**Hosting:** Frontend on Vercel, Backend on Render

## Write-up

### Tech Choices
The frontend is built with React and Vite for fast development and a snappy UI, styled with Tailwind CSS for a clean, emotionally warm design that fits a matrimonial product. The backend is a lightweight Express server using JSON files as a mock database — no SQL overhead for an MVP, easy to seed with realistic data. Authentication uses JWT stored in localStorage with automatic redirect on token expiry. For AI, I chose Groq over OpenAI because it offers a generous free tier with fast inference on Llama 3.1, making it practical for a demo without billing concerns. The architecture keeps frontend and backend fully decoupled, hosted independently on Vercel and Render.

### Matching Logic
The algorithm is gender-specific, reflecting how Indian matrimonial compatibility is realistically assessed. For **male clients**, scoring weights age gap (25 pts — ideal: she is 2–6 years younger), children preference alignment (20 pts), religion/caste (15 pts), height difference (10 pts), income dynamics (10 pts), education tier compatibility (10 pts), and lifestyle habits like diet, smoking, drinking (10 pts), with city and family-type bonuses. For **female clients**, the weights shift to prioritise children alignment (20 pts), shared values — religion and family type (20 pts), professional compatibility via category matching — tech/finance, medical/education, etc. (15 pts), relocation flexibility (15 pts), income (he earns ≥ her, 10 pts), age (he is 1–5 years older, 10 pts), and lifestyle (10 pts). All matches scoring below 30 are filtered out. A quick pre-filter algorithm picks the top 8 candidates by lightweight signals (religion, kids, age gap, city) before sending them to the AI for deep scoring — keeping API calls efficient.

### How AI Is Used
Groq (Llama 3.1 8B Instant) is used in two ways. First, the top 8 pre-filtered candidates per client are sent to the LLM in a single request. The model returns a compatibility score (0–100), a label (High Potential / Strong / Good / Possible / Low), a breakdown across five categories (Family Values, Life Goals, Career, Lifestyle, Practical Fit), specific positives and concerns, and a personalised introduction email for each candidate. This gives matchmakers AI-powered insight beyond rule-based scoring. Second, the Notes Analyser feature lets matchmakers paste raw meeting notes and get structured intelligence back — key preferences, recommended match traits, watch-out concerns, and a suggested next step — all generated by the LLM in real time.

### Assumptions Made
- Each matchmaker only sees clients assigned to them (multi-tenant by `assignedTo` field).
- The match pool (105 male + 105 female profiles) simulates a real database of verified candidates.
- "Send Introduction" triggers a modal showing a pre-generated personalised email — actual email delivery is mocked for the MVP.
- Caste matching is treated as a bonus (not a hard filter) to keep the algorithm inclusive by default.
- The AI layer degrades gracefully — if the Groq API is unavailable, the rule-based algorithm serves all matches without breaking the experience.
- Income figures are annual in INR; profiles span metro and tier-2 Indian cities to reflect TDC's real client base.

## Features

- **Login** — JWT-based matchmaker authentication
- **Dashboard** — Client list with search, status filter, gender filter, list/grid view, and live stats
- **Customer Profile** — Full biodata with 30+ fields across personal, professional, and lifestyle categories
- **Matchmaking Journey Funnel** — Visual stepper (New → Profiling → Searching → Matched → Success)
- **AI Match Scoring** — Top 8 matches deep-scored by Groq with category breakdown bars and personalized explanations
- **Smart Filters** — Filter matches by age range, income, city, religion, kids preference, relocation
- **Send Introduction** — Modal with pre-written AI-generated intro email and one-click send
- **Notes** — Add/delete timestamped notes per client
- **AI Notes Analyser** — Paste meeting notes, get structured insights (preferences, concerns, next steps)
- **Sent History** — Track all introductions sent per client

## Local Setup

```bash
# Install all dependencies
npm run install:all

# Add environment variables
# Create server/.env with:
# PORT=5000
# JWT_SECRET=tdc_matchmaker_super_secret_2024
# GROQ_API_KEY=your_groq_key_from_console.groq.com

# Run frontend + backend together
npm run dev
```
